V.V.C Trench Runner

Collect coins, dodge the traps, and survive the trench. The game can be skinned for each coin, but the core controls stay the same.

Controls

  • Move with WASD or arrow keys.
  • On phones, move with the left touch pad.
  • Pick up coins by touching them.
  • Pick up a gun, then find ammo.
  • Shoot with mouse click, F, Ctrl, or the phone Shoot button. You need both a gun and ammo.
  • You have health now. A run ends when health reaches zero.

Enemies

  • Ruggers chase and end the run if they catch you. Shooting them slows them; enough shots removes them.
  • Snipers aim, then fire. Getting sniped ends the run and applies the snipe penalty.
  • Fuddlers do not end the run. Their bubble slows you down and throws FUD at you.

Specials

  • Honeypots steal the coins you collected during the run if you touch them.
  • Whales chase coins and can grab them before you do.
  • The megacoin is rare and worth a big coin bonus when it appears.
  • Medic bags are rare pickups that restore you to full health.

Token Mode

  • Demo mode is for testing and tuning.
  • Live mode will be the token-cost version once the Solana settlement layer is built.
  • The HUD shows score, collected coins, gun status, ammo, and high score.
